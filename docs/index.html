Ajouts potentielles

  • Incorporer des distributions mélange (comme la Tweedy).
  • Rajouter les mesures de risque (TVaR et VaR) aux distributions discrètes en ajoutant une erreur souhaitée par l'utilisateur.
  • Évaluer comment voir les formules pour les espérance, variance, stop-loss, ... en hoverant au-dessus de ces moments.
  • Ajouter des notes sur les distributions à propos des paramètres utilisés, les fonctions de densité, les liens entre les distributions, les domaines, ...
  • Évaluer comment entrer des fractions comme argument.
  • Évaluer la mise en oeuvre d'un solveur pour les paramètres à partir de données / un GLM ou quelque-chose du genre pour pouvoir déduire les distributions.
  • Évaluer comment inclure un search box pour les distributions dans le menu.
